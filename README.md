# Лабораторные работы №17-№20 — Корякина Анна

**Тема проекта:** Сайт-визитка магазина цветов «Букет настроения».
**Стек:** React 18 + Vite (сборка), HTML + CSS, без CMS — согласно требованиям ТЗ.

🌐 **Демонстрация онлайн:** https://hunterspence.github.io/anna-group-project-5-25/
📦 **Репозиторий:** https://github.com/HunterSpence/anna-group-project-5-25

## Что реализовано

Согласно техническому заданию (см. `Лабораторная_работа_17_20_вариант_студента.docx`):

- ✅ Сайт разработан на **React** (см. `src/`).
- ✅ Шаблон страницы из 3 элементов:
  - **Header** — зелёный, 5% высоты экрана, логотип по центру.
  - **Main** — 89% высоты экрана, делится на три колонки 1:4:1:
    - **Section** — золотой, 20% ширины, сверху ссылки на страницы, снизу баннер.
    - **Article** — синий, 60% ширины, контент текущей страницы.
    - **Aside** — бирюзовый, 20% ширины, сверху облако тегов, снизу баннер.
  - **Footer** — красный, 6% высоты, контактная информация и сведения о создателе.
- ✅ Четыре страницы в меню навигации:
  - «Новости»
  - «Об авторе»
  - «Обратная связь»
  - «Продукты и услуги»
- ✅ Переключение языка интерфейса **русский ↔ английский** (кнопка в правом углу хедера).
- ✅ Возможность скачивать документ (прайс-лист) со страницы «Новости».
- ✅ Форма обратной связи на странице «Обратная связь».
- ✅ Табличная форма с ценами на странице «Продукты и услуги».
- ✅ Облако тегов в правом блоке (aside).
- ✅ Адаптивная вёрстка для разрешений **1920×1080** (десктоп) и **800×600** (мобильное).
- ✅ Текст сайта чёрного цвета, ссылки оформлены по умолчанию.

## Запуск сайта

Требуется **Node.js 18+** ([скачать](https://nodejs.org/)).

```bash
cd Koriakina_lab17-20_flower-shop
npm install
npm run dev
```

Сайт откроется автоматически по адресу **http://localhost:3000** — именно этого
адреса требует тест-кейс из ТЗ.

### Сборка для проверки/демонстрации

```bash
npm run build
npm run preview   # тоже на порту 3000
```

## Тема — флористика

Магазин цветов и букетов «Букет настроения». Контент страниц адаптирован под
тематику магазина: новости — анонсы коллекций и акций, продукты — букеты и
услуги флориста, обратная связь — телефон и форма обращений.
